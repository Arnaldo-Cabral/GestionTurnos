# 🩺 Backend - Gestión de Turnos Médicos
Este proyecto es el backend de una aplicación web profesional para la gestión de turnos e historias clínicas.
## 🚀 Tecnologías
- Node.js + Express
- Sequelize + MySQL
- JWT + bcrypt
- Arquitectura modular
## 📦 Instalación
1. Clonar el proyecto
2. Instalar dependencias:
```bash
npm install
## correr el Backend 
 node server.js
